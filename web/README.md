# Web App

This directory contains the Vite + React frontend for Integrity Dashboard.

## Development

Install dependencies:

```bash
npm install
```

Run the frontend:

```bash
npm run dev
```

The frontend expects the local backend API to be running on `127.0.0.1:3001`. During development, Vite proxies `/api/*` requests to that backend.

## Validation

```bash
npm run lint
npm run build
```
